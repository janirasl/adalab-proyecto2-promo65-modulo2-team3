🎵 Festival Data Beats: Millennial Insights

Data Beats Crew: Fabiola, Dácil, Janira, Natalia y Rocío.

🎯 El Reto
Consultoría estratégica para un promotor de festivales. 

Objetivo: Diseñar un evento internacional para la generación Millennial (2016-2019) con una fuerte identidad de talento español.

Géneros: Latin, Reggaeton, Hip-hop, Indie y Rock.

🛠️ Tecnología y Metodología
Agile: Gestión con Scrum.

Datos: API de Last.fm y archivos JSON.

Stack: Python (Pandas, BeautifulSoup para limpieza de HTML, SQLAlchemy) y SQL para el análisis.

Seguridad: Uso de .env para protección de credenciales.
